## Steps
1. **Planning and Image Prep** | I began by selecting a few pieces of artwork I felt proud of and photographs I took over the time. Only a few were selected in order to avoid an overwhelming amount in the portfolio. I used Adobe Photoshop to edit and optimize each asset. I then moved to Figma to finalize the layouts, color palette, and overall aesthetic of the page. [View the Figma Design Mockup Here](https://www.figma.com/design/pOatk8BuwGV3JzZJzOqvSV/Art-Portfolio-Planning?node-id=0-1&t=1QENJkhol2IWBFT0-1)
   
2. **Building the Coding Fundation** | I then proceeded with the coding foundation using HTML and React. I wrote the skeleton of the website that consisted of a simple structure of a one-page website. This allows for easy navigation and a compact feel.
   
3. **Setting Up the Enviroment** | I have been using Vite to run the webpage locally which has helped vision the future of the page and a fast development environment.
   
4. **Creativity with CSS** | I have been working on designing the look with CSS. This has been the most challenging aspect of the project but have enjoyed seeing what creativity aspects coding can take part of.

5. **Future Goals** | While it is still a work in progress, I will continue this project and continue to implement the features I have planned. My next steps involve implementing the final image galleries and ensuring full responsiveness before the final deployment.


## How to Run this Project
1. Clone the repository | Copy the URL (https://github.com/acama5/art-portfolio.git) and run the git clone command in your       terminal.

2. Navigate into the project directory | Use the command "cd art-portfolio" to enter the folder.

3. Install dependencies | Run "npm install" to download the project files if needed.

4. Launch the Vite | Run "npm run dev" to start the project then open the local link!
